Payment Reminder Voice Agent

a compliant outbound calling agent
LiveKit Agents
LiveKit · infrastructureThe real-time transport and agent runtime. It bridges the phone call into a room, hosts the agent worker, and runs the full STT→LLM→TTS session — everything inside this box lives on LiveKit.
infrastructure · transport & session
Guardrails guardrails.py
Guardrails · compliance shellWraps the entire reasoning core. Blocks the call pre-flight on an active grievance, and screens every utterance — dispute, hardship, wrong person, "stop calling", human requests — halting the payment flow and catching prohibited language before it is spoken.
State Machine state_machine.py
State Machine · call flowTwelve call states from opening disclosure through identity check, payment context, promise-to-pay and handoff. Each state defines exactly which actions and tools the model is allowed to use next.
Prompt
Prompt · system instructionsRebuilt for the current state on every turn. Injects the borrower context and the state's allowed actions into the system prompt so the model only ever knows what it is permitted to do right now.
Borrower
The BorrowerThe agent dials them — this is an outbound call. They answer, speak, and hear the agent's reply. Identity is verified before any account detail is disclosed.
Payment Agentrun.py · worker
Payment Agent · orchestratorThe agent worker (agent.py + run.py) running on LiveKit. It dispatches the outbound call, drives the STT→LLM→TTS session, and enforces the state machine and guardrails on every turn.
TwilioPSTN · SIP
Twilio · telephonyElastic SIP trunk that carries the outbound call over the PSTN and supplies the caller ID. Also sends post-call WhatsApp confirmations.
Speech → Textdeepgram · whisper
STT · speech to textStreaming transcription — Deepgram nova-3 by default, OpenAI Whisper as an alternative. Silero VAD gates the mic so the agent only transcribes real speech.
LLMgpt-4o-mini · gemini
LLM · the reasoning coreReads the transcript, decides which tool to call, and drafts the reply. Pluggable — OpenAI or Gemini via one env var. On its own it is unconstrained; the three layers around it make it compliant.
Murftext → speech
Murf · text to speechStreams the LLM's reply text into natural-sounding speech in the configured agent voice, sent back through LiveKit and Twilio to the borrower.
